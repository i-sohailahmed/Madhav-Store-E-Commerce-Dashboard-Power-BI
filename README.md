#  Madhav Store E-Commerce Dashboard | Power BI

##  Project Overview

This project is an interactive Power BI dashboard built to analyze sales performance for **Madhav Store**, an e-commerce business.

The dashboard provides insights into revenue, profit, customer purchasing behavior, product categories, payment methods, and monthly business performance. It allows users to interactively filter the data by **State** and **Quarter** to explore business trends.

---

#  Business Objective

The objective of this dashboard is to help business stakeholders:

- Monitor sales performance
- Track profit trends
- Identify top-performing states
- Understand customer purchasing behavior
- Analyze payment preferences
- Discover profitable product categories
- Support data-driven business decisions
